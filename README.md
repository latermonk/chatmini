# 微信小程序图书管理系统

一个功能完善的微信小程序图书管理系统,支持图书入库、借出、归还等核心功能。

## 功能特点

### 1. 首页
- 统计展示:图书总数、可借阅数量、已借出数量
- 快捷操作:扫码添加图书、手动添加图书
- 最近添加:显示最近入库的5本图书

### 2. 图书列表
- 全部图书浏览
- 按状态筛选:全部/可借阅/已借出
- 关键词搜索:支持按书名或作者搜索
- 点击查看图书详情

### 3. 图书详情
- 完整图书信息展示:书名、作者、ISBN、出版社、出版日期、存放位置、简介等
- 借出功能:输入借阅人姓名,记录借阅时间
- 归还功能:记录归还时间
- 编辑功能:修改图书信息
- 删除功能:删除未借出的图书
- 当前借阅信息展示

### 4. 借阅记录
- 全部借阅记录浏览
- 按状态筛选:全部/借阅中/已归还
- 显示借阅人、借阅时间、归还时间等信息
- 点击可跳转到对应图书详情

### 5. 添加图书
- 手动添加:填写完整的图书信息
- 扫码添加:扫描ISBN条码快速录入
- 支持上传封面图片
- 表单验证:必填字段检查

### 6. 扫码功能
- 自动扫描ISBN条码
- 扫描结果展示
- 快速跳转到添加页面填写信息

## 技术特点

- 纯前端实现,使用微信小程序本地存储
- 响应式数据绑定
- 完整的页面生命周期管理
- 友好的用户交互体验
- 数据持久化存储

## 页面结构

```
├── pages/
│   ├── index/           # 首页
│   ├── book-list/       # 图书列表
│   ├── book-detail/     # 图书详情
│   ├── borrow-list/     # 借阅记录
│   ├── add-book/        # 添加/编辑图书
│   └── scan-code/       # 扫码添加
├── images/              # 图片资源
├── app.js              # 小程序逻辑
├── app.json            # 小程序配置
├── app.wxss            # 小程序样式
└── sitemap.json        # 索引配置
```

## 数据结构

### 图书数据
```javascript
{
  id: 'book_1',              // 图书唯一ID
  title: '图书标题',         // 书名
  author: '作者',           // 作者
  isbn: '978-7-xxx',        // ISBN号
  publisher: '出版社',       // 出版社
  publishDate: '2024-01-01', // 出版日期
  location: '书架A-1',      // 存放位置
  cover: '/images/xxx.png', // 封面图片
  description: '图书简介',   // 简介
  status: 'available',      // 状态: available(可借)/borrowed(已借)
  addTime: '2024-01-01 10:00' // 入库时间
}
```

### 借阅记录
```javascript
{
  id: 'record_1',           // 记录ID
  bookId: 'book_1',        // 图书ID
  bookTitle: '图书标题',   // 图书标题
  borrower: '张三',        // 借阅人
  borrowTime: '2024-01-01 10:00', // 借阅时间
  returnTime: '2024-01-10 10:00', // 归还时间
  status: 'returned'       // 状态: borrowed(借阅中)/returned(已归还)
}
```

## 使用说明

1. 在微信开发者工具中打开项目
2. 点击"编译"按钮预览
3. 在真机上扫码体验完整功能

## 注意事项

- 本项目使用本地存储,数据保存在小程序本地
- 如需云端存储,可接入微信云开发
- 扫码功能需要在真机上测试,模拟器可能不支持
